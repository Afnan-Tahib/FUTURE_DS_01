# Sales & Profit Performance Dashboard

## Overview
This project analyzes business sales performance using Tableau.

## Key Insights
- Technology is the highest revenue and profit generating category.
- Furniture contains loss-making sub-categories such as Bookcases and Tables.
- West region performs best in profitability.
- Central region shows improvement potential.

## Tools Used
- Tableau Public
- Superstore Dataset
